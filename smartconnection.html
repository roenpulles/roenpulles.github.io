Smart connection lessen

Bij de lessen smart conection werd er uitgelegd hoe je een website moet maken.
Bij de minor Smart Industry worden er geen tentamens gegeven maar word je uiteindelijk beoordeeld op je portfolio.
Je persoonlijke portfolio laat je deze keer via een website zien. Ik had hiervoor nog nooit ervaring met websites maken dus dit was voor mij een hele nieuwe wereld.
Om de website te maken moesten we aan de slag met html coderen. Via Youtube waren er filmpjes beschikbaar die meer info gaven over html en css.
Deze filmpjes waren ervoor bedoeld om de basis te leren. In je website willen ze graag zien wat je dit semester allemaal hebt geleerd. En één van die geleerde dingen is de website zichzelf.
Natuurlijk kan je kant en klare templates downloaden en zal daardoor ook je website professioneler maar het leek mij juist leuker om iets te leveren waarvan je kan zien dat echt zelf gemaakt is.
Ik begon met een hele simpele basis website en naarmate de tijd werd die steeds uitgebreider. Ik heb vaak vastgezeten en ook vaak navraag aan mede studenten gedaan hoe zij daar uit kwamen. Ook was de site W3SCHOOLS een handige tool om bepaalde widgets in je website te maken. Je kunt bijvoorbeeld op mijn homepagina op de Youtube/Linkedin button klikken om zo naar mijn account verwezen te worden.



















Gedachtes achter mijn website

Natuurlijk is elke website persoonlijk maar ik wilde mijn ook met kleine verwijzing naar mijn hobby. Ik studeer zelf aan de Automotive en heb altijd al een passie voor auto´s gehad.
Dit wilde ik dus graag terug laten komen. Mijn achtergrond is daardoor ook een autoweg, ook als je op de Volvo C30 linksboven klikt ga je weer terug naar de voor pagina.
Het is dus eigenlijk een home button maar dat met een mooi verwijzing naar mijn passie. Ik ben ook een Volvo liefhebber dus vandaar dat je bij mijn website tablad het Volvo logo ziet staan.


Wat heb ik geleerd en wat valt er nog te leren

Ik heb bij dit vak dus geleerd om een website te kunnen programeren, en de basis geleerd van hoe nou een website in elkaar zit. Er valt alleen nog veel meer te leren.
Ik zelf vind de opmaak van mijn website er keurig uitzien maar had nog veel uitgebreider gekund. Maar voor wat er op geleverd moet worden vind ik dit een keurige oplevering.
Ik ben ook blij dat ik alles zelf heb gedaan en niet zomaar een template heb gedownload van het internet, daarvan leer je niks en op de manier wel.
